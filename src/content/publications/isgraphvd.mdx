## Overview

This work targets vulnerability detection in IoT-oriented software supply chains. It emphasizes identifier-sensitive graph construction so that analysis can better preserve the semantic relations needed for precise detection.

## Research context

The paper sits at the intersection of supply chain security, program analysis, and trustworthy software engineering, complementing Ziang Liu's work on compliance analysis with security-oriented dependency reasoning.
